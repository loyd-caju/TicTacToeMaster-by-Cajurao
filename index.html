How to Play

  • Click/tap an empty cell to place your mark.
  • Win the round by getting 3 in a row.
  • The match ends when someone reaches the selected Match Length (First to 3/5/7).

Difficulty (Vs Computer)

  • Easy: mostly random moves.
  • Normal: mixes smart + random (beat it with good reads).
  • Hard: plays optimal (expect pain 😅).

Keyboard Controls

  • Arrow Keys move selection
  • Enter / Space place mark
  • R reset current round
  • N start a new match (scores reset)
